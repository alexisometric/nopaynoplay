# Politique de sécurité

## Versions supportées

Seule la dernière version mineure est activement supportée.

| Version | Supportée |
|---|---|
| 1.0.x   | ✅ |
| < 1.0   | ❌ |

## Signaler une vulnérabilité

**N'ouvre pas d'issue GitHub publique** pour une faille de sécurité.

Utilise plutôt l'une de ces options :

1. [GitHub Security Advisories](https://github.com/alexisometric/nopaynoplay/security/advisories/new) (recommandé — chiffré, traçable)
2. Contacter directement le mainteneur en privé sur son profil GitHub.

Inclus si possible :

- une description claire de la faille,
- les étapes pour la reproduire,
- l'impact estimé (qui peut être affecté, comment),
- une éventuelle proposition de correctif.

Tu recevras une réponse sous **72 heures**. Un correctif sera publié dès que possible, et un crédit te sera attribué dans les notes de release (sauf demande contraire).

Merci de contribuer à rendre NoPayNoPlay plus sûr 🙏
